MMA Website Project


MMA wiki with updated data at the time of creation. Built with HTML and CSS. Had a blast making it. Work was made in a group of 2 people. As you might have guessed, I was one of them.

As fans of the sport, we identified a shortage of websites that prioritised finding the relevant information regarding fighters, events and rankings in a fast and efficient way. Typically, these things require using multiple websites to gather. We created a solution for this problem, where each step in a typical use-case is one click away.

Time to complete:

3 months

Java Project

A java project representing a dog register, where the user is able to register dogs and owners, and giving/removing said dogs to/from the owners, while maintaining the correct relationship between the two. Program is mostly object-oriented. Every dog that is registered has its own name, age, and physical attributes.

The project was created using IDEA IntelliJ and is related to a Java programming course from Stockholm University. The work was done individually.

Time to complete:

6 months
